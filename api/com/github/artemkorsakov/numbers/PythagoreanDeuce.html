Companion object PythagoreanDeuce

case class PythagoreanDeuce(m: Long, n: Long) extends Product with Serializable

A Pythagorean triplet is a set of three natural numbers, a < b < c, for which, a2 + b2 = c2, a=m2 - n2, b=2*m*n, c=m2 + n2. (m,n) - Pythagorean deuce.
